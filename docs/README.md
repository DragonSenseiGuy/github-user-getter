# Github User Getter

A simple Flask application to find and display GitHub user profiles, including their contribution graph and pinned repositories.

## Features

- **User Search:** Search for any GitHub user by username.
- **Profile Overview:** View user details like bio, followers, following, and location.
- **Contribution Graph:** Visual representation of the user's contributions.
- **Pinned Repositories:** Displays the user's pinned repositories with star counts and language.
- **Responsive Design:** Built with Bulma CSS for a clean and mobile-friendly interface.
