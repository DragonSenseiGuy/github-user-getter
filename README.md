# GitHub User Getter
Great name I know :), this is a Flaks and Bulma application where you enter a persons gitHub username and you can get their github info.

## Why?
Q: Why should I use this when I can go to their profile and see their info too?

## Installation
First make a virtual environment using:
```commandline
python -m venv .venv
```
to activate on macOS/Linux run:
```commandline
source .venv/bin/activate
```

on Windows run:
```commandline
venv\Scripts\activate.bat
```

then install dependencies using
```commandline
pip install -r requirements.txt
```

now run the project using
```commandline
python app.py
```
that's it!
